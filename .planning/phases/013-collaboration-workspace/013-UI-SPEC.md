## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Inline icon/text gaps, live cursor offsets |
| sm | 8px | Kanban card internal padding, notification item gaps |
| md | 16px | Standard padding for Kanban columns and Notification panels |
| lg | 24px | Spacing between Editor toolbar and content |
| xl | 32px | Section gaps (e.g., between timeline and Kanban board) |
| 2xl | 48px | Major layout separations |
| 3xl | 64px | Page margins |

Exceptions: none

| Label | 14px | Medium (500) | 1.4 |
| Heading | 24px | SemiBold (600) | 1.2 |
| Display | 32px | Bold (700) | 1.1 |

---

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #FFFFFF / #0F172A | Document editor background, main workspace surface |
| Secondary (30%) | #F8FAFC / #1E293B | Kanban board columns, notification panels, timeline |
| Accent (10%) | #3B82F6 | Primary buttons, active presence rings, unread badges |
| Presence Colors | Variable | Each user gets a deterministic distinct color (e.g., #EC4899, #8B5CF6) for their live cursor and presence indicator avatar ring. |
| Destructive | #EF4444 | Deleting tasks or documents |
