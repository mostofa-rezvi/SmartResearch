---

## Component-Specific Contracts

| UI Element | Visual & Interaction Contract |
|------------|-----------------------------|
| **Journal recommender results** | Displayed in a responsive grid of cards. Each card emphasizes impact factor and relevance score. Hover states elevate the card (`shadow-md`). |
| **Publication checklist UI** | Vertical layout with checkbox components. Checked items become slightly transparent with a strikethrough text effect. Progress bar at the top updates dynamically. |
| **Forum thread layout** | Hierarchical left-bordered nesting for replies. Avatar and TrustRank badges are pinned to the top-left of each message. |
| **TrustRank badges** | Small pill shape (px-2 py-0.5), font-medium, text-xs. Gold/Silver/Bronze color coding depending on the rank level. |
| **Spam filter status** | Embedded alert components at the top of a thread or post. Uses destructive red color with an alert icon (e.g., triangle warning). |
