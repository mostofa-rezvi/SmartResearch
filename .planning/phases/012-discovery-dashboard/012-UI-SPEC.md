## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing |
| md | 16px | Default element spacing (e.g. between sidebar items) |
| lg | 24px | Section padding (e.g. inside collaborator cards) |
| xl | 32px | Layout gaps (e.g. between sidebar and main feed) |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions: none

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #FFFFFF / #0F172A | Background, main feed surface |
| Secondary (30%) | #F8FAFC / #1E293B | Sidebar backgrounds, search bar, collaborator cards |
| Accent (10%) | #3B82F6 | Primary buttons, active filters, similarity score badges |
| Destructive | #EF4444 | Destructive actions only |

Accent reserved for: Active filter indicators, similarity score highlights, primary CTA buttons (e.g. "Connect", "Follow"). TrustRank tiers will use a distinct metallic scale (Bronze, Silver, Gold) but primary interactive elements remain the Accent color.
