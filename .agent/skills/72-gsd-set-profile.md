# /gsd-set-profile

**Category:** 1. Project Initialization & Planning  
**Phase:** Anytime — agent configuration

## Description

Switch model profile for GSD agents (quality/balanced/budget/inherit).

## When to Use

- Use `quality` for critical phases (auth, matching engine, security)
- Use `balanced` for standard build phases
- Use `budget` for simple config tasks and housekeeping

## Command

```
/gsd-set-profile quality
/gsd-set-profile balanced
/gsd-set-profile budget
```
