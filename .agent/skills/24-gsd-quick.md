# /gsd-quick

**Category:** 3. Execution & Building  
**Phase:** Medium tasks

## Description

Execute a quick task with GSD guarantees (atomic commits, state tracking) but skip optional agents.

## When to Use

- Medium tasks: add a new route, create a middleware
- Write a single service function
- Tasks that need GSD tracking but not full planning

## Command

```
/gsd-quick
```
