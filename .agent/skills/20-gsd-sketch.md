# /gsd-sketch

**Category:** 2. Phase Planning & Discussion  
**Phase:** Before UI implementation

## Description

Sketch UI/design ideas with throwaway HTML mockups, or propose what to sketch next (frontier mode).

## When to Use

- Before committing to UI implementation
- Quick visual prototyping of dashboard layouts, search UX, profile pages
- Throwaway mockups to validate design direction

## Command

```
/gsd-sketch
```
