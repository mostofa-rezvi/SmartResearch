# /gsd-remove-phase

**Category:** 2. Phase Planning & Discussion  
**Phase:** When phases become unnecessary

## Description

Remove a future phase from roadmap and renumber subsequent phases.

## When to Use

- When a planned phase becomes unnecessary
- E.g., Kafka setup phase removed since we're using Redis instead
- Duplicate or superseded work identified

## Command

```
/gsd-remove-phase
```
