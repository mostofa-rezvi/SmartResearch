# /gsd-forensics

**Category:** 5. Debugging & Troubleshooting  
**Phase:** After failed workflows

## Description

Post-mortem investigation for failed GSD workflows — analyzes git history, artifacts, and state to diagnose what went wrong.

## When to Use

- When a phase execution fails unexpectedly
- Analyzes git history, artifacts, and state
- Produces root cause diagnosis

## Command

```
/gsd-forensics
```
