# /gsd-validate-phase

**Category:** 4. Quality Assurance & Review  
**Phase:** Retroactive quality check

## Description

Retroactively audit and fill Nyquist validation gaps for a completed phase.

## When to Use

- Retroactive quality check for phases executed quickly
- Fill validation gaps that were skipped
- Ensure coverage meets minimum quality bar

## Command

```
/gsd-validate-phase
```
