# /gsd-plan-milestone-gaps

**Category:** 7. Milestone Management & Shipping

## Description

Create phases to close all gaps identified by milestone audit.

- Auto-generates new phases to fill gaps
- End of Week 1, 2, or 3 if audit reveals missing work

## Command

```
/gsd-plan-milestone-gaps
```
