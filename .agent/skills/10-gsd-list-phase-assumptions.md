# /gsd-list-phase-assumptions

**Category:** 2. Phase Planning & Discussion  
**Phase:** Before complex phases

## Description

Surface the agent's assumptions about a phase approach before planning.

## When to Use

- Before complex phases like the ML matching engine or TrustRank
- Catch wrong assumptions early before they propagate into plans
- Validate technical approach before committing resources

## Command

```
/gsd-list-phase-assumptions
```
