# /gsd-next

**Category:** 7. Milestone Management & Shipping  
**Phase:** When unsure what to do next

## Description

Automatically advance to the next logical step in the GSD workflow.

## When to Use

- When you're unsure what to do next
- GSD figures out the right action based on current state
- Routes to discuss, plan, execute, or review

## Command

```
/gsd-next
```
