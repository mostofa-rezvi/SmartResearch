# /gsd-ingest-docs

**Category:** 1. Project Initialization & Planning  
**Phase:** When importing external documentation

## Description

Scan a repo for mixed ADRs, PRDs, SPECs, and DOCs and bootstrap or merge the full `.planning/` setup from them.

## When to Use

- If you have existing design docs, specs, or decision records
- Import into the GSD workflow from external sources
- Classifies each doc, synthesizes consolidated context
- Routes to new-project or merge-milestone depending on `.planning/` state

## Command

```
/gsd-ingest-docs
```
