# /gsd-scan

**Category:** 1. Project Initialization & Planning  
**Phase:** Before starting new work

## Description

Rapid codebase assessment — lightweight alternative to `/gsd-map-codebase`.

## When to Use

- Quick health check of existing code before starting new work
- Understand current project state without deep analysis
- Verify what's already built (auth, controllers, services, Docker config)

## Command

```
/gsd-scan
```
