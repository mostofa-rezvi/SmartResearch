# /gsd-progress

**Category:** 7. Milestone Management & Shipping  
**Phase:** Daily check

## Description

Check project progress, show context, and route to next action (execute or plan). Use `--forensic` to append a 6-check integrity audit.

## When to Use

- Daily check: what's done, what's next, any blockers
- Route to next action automatically
- `--forensic` for deeper integrity audit

## Command

```
/gsd-progress
/gsd-progress --forensic
```
