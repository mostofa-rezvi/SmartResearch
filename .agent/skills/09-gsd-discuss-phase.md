# /gsd-discuss-phase

**Category:** 2. Phase Planning & Discussion  
**Phase:** Before planning each day's work

## Description

Gather phase context through adaptive questioning before planning.

## When to Use

- Before planning each day's work
- Use `--auto` to skip interactive questions (agent picks defaults)
- Use `--chain` for interactive discuss → automatic plan + execute
- Use `--power` for bulk question generation into file-based UI
- Use `--all` to skip area selection and discuss all gray areas

## Command

```
/gsd-discuss-phase
/gsd-discuss-phase --auto
/gsd-discuss-phase --chain
/gsd-discuss-phase --power
```
