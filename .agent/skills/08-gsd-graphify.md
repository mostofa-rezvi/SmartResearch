# /gsd-graphify

**Category:** 1. Project Initialization & Planning  
**Phase:** After phases are defined

## Description

Build, query, and inspect the project knowledge graph in `.planning/graphs/`.

## When to Use

- Visualize dependencies between phases, services, and data flows
- Architecture clarity for complex multi-service systems
- Understand how backend ↔ ML service ↔ databases interact

## Command

```
/gsd-graphify
```
