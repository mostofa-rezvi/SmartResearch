# /gsd-undo

**Category:** 5. Debugging & Troubleshooting  
**Phase:** After bad commits

## Description

Safe git revert. Roll back phase or plan commits using the phase manifest with dependency checks.

## When to Use

- Roll back a bad phase execution or broken commit
- Without disrupting dependent work
- Checks dependencies before reverting

## Command

```
/gsd-undo
```
