3.  Technology Stack Rationale
Layer
	Technology
	Why This Choice
	Frontend
	Next.js (React)
	SSR = SEO-indexable academic content; faster page load
	UI
	Tailwind CSS
	Utility-first; zero runtime overhead; rapid iteration
	Real-time
	Socket.IO
	WebSocket + auto-fallback; collaborative editing & live notifications
	API Gateway
	Node.js + Express
	Non-blocking I/O; handles 10k+ concurrent connections efficiently
	ML Service
	Python + FastAPI
	ASGI async; native Python ML ecosystem; auto-generates OpenAPI docs
	Matching Algo
	CF + CBF Hybrid
	CBF solves cold-start; CF improves with user behaviour over time
	Semantic Search
	Sentence-BERT
	Dense embeddings understand meaning, not just keywords
	Trust Scoring
	TrustRank (PageRank)
	Credibility flows from verified professors; resists spam/gaming
	Primary DB
	PostgreSQL
	ACID-compliant; handles complex relational queries; battle-tested
	Graph DB
	Neo4j
	Multi-hop traversal (2nd-degree collaborators) in milliseconds
	Search Index
	Elasticsearch
	Unified BM25 + kNN vector search across people, papers, posts
	Cache
	Redis
	Pre-computes recommendation feed as background job; <50ms load
	Message Queue
	Apache Kafka
	Decouples API from ML workers; async processing; durable log
	Infrastructure
	Docker + Kubernetes
	Per-service scaling; ML pods scale independently on load spikes
	

4.  Impact Summary — Problem vs. Solution
#
	Problem
	ResearchBridge Solution
	Measurable Outcome
	1
	No structured collaborator discovery tool
	CF+CB Hybrid + Sentence-BERT surfaces collaborators from profile & behaviour signals
	Matched collaborators on first login; relevance improves with use
	2
	No shared workspace — ad-hoc email/Drive/WhatsApp
	Built-in workspace: task board, milestone tracker, real-time doc co-editing
	All project activity in one place; version history; no context lost
	3
	New researchers don't know how to publish
	Publication Assistant: journal recommender (DOAJ/Scimago) + templates + checklist
	Reduces 'I don't know where to submit' from weeks to minutes
	4
	Community forums decay into spam
	TrustRank algorithm: PageRank-variant weights credibility by citations & endorsements
	Quality-first feed; verified institutional badges auto-assigned
	5
	Cold-start: new users get no useful recommendations
	Hybrid approach: CBF bootstraps from explicit tags; CF activates as behaviour accumulates
	Even 0-paper researchers get relevant results from day one
